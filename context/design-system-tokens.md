# Design System Tokens

> Preencha este arquivo com os tokens e componentes principais do
> design system do Loft. Os agentes usam isso para validar uso correto
> de componentes e tokens durante a revisão de jornadas.

---

## Tokens de Cor

### Cores Primárias
| Token | Valor | Uso |
|-------|-------|-----|
| `color-primary-default` | #[hex] | [CTA principal, links] |
| `color-primary-hover` | #[hex] | [Estado hover do CTA] |

### Cores de Feedback
| Token | Valor | Uso |
|-------|-------|-----|
| `color-success` | #[hex] | [Confirmações, sucesso] |
| `color-error` | #[hex] | [Erros, atenção crítica] |
| `color-warning` | #[hex] | [Alertas, atenção] |

### Cores Neutras
| Token | Valor | Uso |
|-------|-------|-----|
| `color-neutral-100` | #[hex] | [Background principal] |
| `color-neutral-900` | #[hex] | [Texto principal] |
