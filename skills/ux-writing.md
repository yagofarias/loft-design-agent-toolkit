# Skill: Copy Review

## Propósito

Copy de interface é design. Uma label ambígua causa suporte. Um erro genérico gera frustração. Esta skill fornece critérios objetivos para avaliar e melhorar textos de interface.

## Princípios de UX Writing

- **Clareza antes de criatividade** — Um CTA que o usuário entende na primeira leitura é melhor do que um CTA espirituoso que gera dúvida
- **Contexto muda o tom** — "Ops, algo deu errado" pode funcionar em um contexto leve mas é inadequado para mensagem sobre pagamento negado
- **Especificidade gera confiança** — "Não foi possível conectar ao servidor de pagamentos" é melhor do que "Ocorreu um erro"
- **Ativo é melhor do que passivo** — "Salvar alterações" é melhor do que "Alterações podem ser salvas"

## Critérios por Tipo de Copy

### CTAs

**Padrão:** `[Verbo] + [Objeto]` quando possível
- ✅ "Criar conta gratuita"
- ✅ "Enviar pedido"
- ❌ "Confirmar" (confirmar o quê?)
- ❌ "Avançar" (para onde?)

### Mensagens de Erro

**Estrutura:** `[O que aconteceu] + [Por que] + [O que fazer]`
- ✅ "Não conseguimos processar o pagamento. Verifique os dados do cartão e tente novamente."
- ❌ "Erro de pagamento. Tente novamente."

### Empty States

**Estrutura:** `[Por que está vazio] + [O que fazer para mudar isso]`
- ✅ "Você ainda não tem pedidos. Explore o catálogo para fazer seu primeiro pedido."
- ❌ "Nenhum resultado encontrado."

## Revisão de Tom por Momento

| Momento da jornada | Tom ideal |
|-------------------|-----------|
| Onboarding / Boas-vindas | Acolhedor, animador, claro |
| Uso recorrente | Direto, eficiente, respeitoso |
| Estado de erro | Neutro, informativo, orientador |
| Ação destrutiva | Sério, preciso, sem ambiguidade |
| Comunicação financeira | Formal, preciso, sem informalidade |

## Checklist Rápido

- [ ] Está em português brasileiro sem anglicismos desnecessários?
- [ ] O verbo está na forma imperativa para CTAs?
- [ ] Evita pontuação desnecessária?
- [ ] O comprimento é adequado ao espaço disponível?
- [ ] É consistente com o guia de voz do produto?
- [ ] Faz sentido fora do contexto visual?
